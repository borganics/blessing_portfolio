## ✨ Features

- **Dynamic Hero Section** – Typewriter effect, floating particles, and a rotating image frame.
- **14+ Mastered Tools** – Google Workspace, Notion, Trello, Canva, Slack, Zoom, Asana, ClickUp, Monday.com, HubSpot, Calendly, Toggl, World Time Buddy, EHR Systems.
- **Medical Services** – 9 comprehensive services for healthcare professionals.
- **Experience Timeline** – 4 professional roles with detailed descriptions and skills.
- **Case Study Modals** – Interactive cards that open detailed project breakdowns:
  - Webinar Launch System (Trello)
  - Team Workspace Setup (Slack)
  - Project Tracking System (Google Sheets)
- **Auto-Smooth Carousels** – Seamless, continuous sliding carousels for:
  - **Work Samples** (9 unique project images)
  - **Testimonials** (8 client reviews)
- **Visual Enhancements**:
  - Floating particles & geometric shapes for depth.
  - Custom cursor & glow trail (desktop).
  - Noise texture overlay for a premium feel.
  - 3D tilt effect on cards.
- **Social Media Preview (Open Graph / Twitter Cards)** – Shares a rich preview with your profile photo, title, and description on LinkedIn, X, Facebook, and WhatsApp.
- **Fully Responsive** – Optimised for desktop, tablet, and mobile devices.
- **Contact Form** – Integrated with Formspree (`https://formspree.io/f/mlgqbzoq`) for easy messaging.
- **Booking Integration** – Direct Calendly scheduling link for discovery calls.
- **Download Resume** – One-click download for CV/Resume.

## 🚀 Getting Started (Local Development)

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/borganics/blessing_portfolio.git
   cd blessing_portfolio
Add your images
Place your profile photo and work sample images into the assets/ folder.
(Make sure the filenames match those referenced in index.html).

Open the site
You can simply open index.html in your browser. For the best experience (to avoid CORS issues with fonts or icons), use a local development server:

Using VS Code – install the Live Server extension and click "Go Live".

Using Python – python -m http.server 8000 and visit http://localhost:8000.

🌐 Deployment (GitHub Pages)
This repository is configured for GitHub Pages. To deploy your live site:

Push your code to the main branch.

Go to your repository on GitHub → Settings → Pages.

Under Branch, select main and save.

After a few minutes, your site will be live at:
https://borganics.github.io/blessing_portfolio/

📝 Customisation Tips
Change the profile image – Replace assets/blessing.jpg with your own photo.

Update social links – The contact section already includes all links. Update the href attributes in index.html if needed.

Modify the typewriter phrases – Edit the phrases array in script.js (currently: Handle Administrative Tasks, Focus on Patient Care, Streamline Your Practice, Reclaim Your Time, Simplify Your Workflow).

Adjust the carousel speed – Look for 3500 (work samples) and 4000 (testimonials) in script.js to change the slide interval.

Update the Formspree endpoint – The form currently submits to https://formspree.io/f/mlgqbzoq. Replace it with your own Formspree ID if needed.

Update Calendly link – The booking button links to https://calendly.com/blessingibeh736/strategy-discovery-call. Update with your own Calendly URL.
